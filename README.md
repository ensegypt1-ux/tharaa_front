# لوحة تحكم سوق ثراء

لوحة تحكم عربية بالكامل (RTL) مرتبطة بخلفية Tharaa Market API V1.

## التشغيل

```bash
cd tharaa_market_admin
npm install
npm run dev
```

افتح: http://localhost:3001

## البيئة

```
NEXT_PUBLIC_API_BASE_URL=http://192.168.10.37:3000/api/v1
```

للبحث عن صور المنتجات عبر Pexels، عيّن `PEXELS_API_KEY` في ملف `.env` الخاص بالخادم فقط.
